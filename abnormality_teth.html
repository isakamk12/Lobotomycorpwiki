ABNORMALITY ENCYCLOPEDIA - TETH

TETH

ZAYINから一歩進み、管理方法を間違えると職員が容易に死亡したり、脱走して施設に被害をもたらすようになる危険度です。しかし抽出できるE.G.Oの性能も大きく向上しています。

要注意・初見殺しギミック

TETH

血の風呂

T-05-41

  • 管理の罠と注意点: 「勇気ランク1」の職員が作業すると、湯船に引きずり込まれて即死します。勇気ランク2以上の職員であれば無害です。
TETH

母なるクモ

T-02-43

  • 管理の罠と注意点: 蜘蛛の巣が張られた部屋を掃除しようとする「洞察作業」を行うと、繭にされて即死します。洞察作業さえ避ければ優秀なレベリング要員です。
TETH

美女と野獣

F-01-57

  • 管理の罠と注意点: 「抑圧作業」を2回連続で行うと、その職員が新たな美女と野獣へと変異（即死）してしまいます。抑圧とそれ以外の作業を交互に行う必要があります。
TETH

壊れゆく甲冑

O-05-61

  • 管理の罠と注意点: 抑圧作業を行うと正義ステータスが上昇するギフトをもらえますが、このオーラを持った職員が「愛着作業」を行うと、打ち首にされて即死します。
  • 武器「決死の一生」 (HEに格上げ): Pale / 近接武器。非常に希少なペイル属性（最大HPに対する割合ダメージ）を持ち、中盤以降も長く使える名刀です。

優秀なE.G.O・要注意バグ

TETH WARNING: 睡眠バグ

空虚な夢

T-02-71

  • 致命的バグ: 脱走時に放つ範囲睡眠の光線を受けた職員が、「クリックしても起きない、または制御不能が解除されない」というバグが頻発します。このまま1日を終えると死亡扱いになるため、絶対に脱走させてはいけない死神です。
TETH WARNING: 反射事故

罰鳥

O-02-56

  • 管理の罠と注意点: 脱走時の被害は小さいですが、こちらから攻撃（反撃）すると1000の即死ダメージを与えてきます。「ブラックスワン」などの自動反撃ダメージ装備をしていると、勝手に怒らせてカウンターで即死する理不尽事故が起きます。
  • 武器「くちばし」: Red / 拳銃。TETH拳銃三銃士の1つ。
TETH WARNING: ダメージ不発バグ

キュートちゃん

F-02-70

  • 武器の罠: 武器「可愛い」は本来1振りで2回攻撃する設定ですが、バグのため攻撃判定が1回しか発生しません。DPSが初期装備「警棒」と完全に一致するTETH最弱の罠武器です。
TETH

捨てられた殺人者

T-01-31

  • 武器「後悔」: Black / ハンマー。攻撃速度は遅いものの、TETH近接武器の中で最高クラスのDPSを誇り、範囲攻撃特性を持つため集団戦で頼りになります。
TETH

マッチガール

F-01-02

  • 武器「4本目のマッチの火」: Red / キャノン。遠距離からの単発高火力と、射線上の敵全てに当たる範囲攻撃（貫通）特性を持ちます。TETH遠距離武器でぶっちぎりのDPS1位です。
TETH

壁に向かう女

F-01-18

  • 武器「悲鳴」 (HEに格上げ): White / クロスボウ。貫通能力を持つ優秀な武器。説明文には「勇気と慎重が3未満だとMP減少」とありますが、バグのためデメリットが機能していません。低ステータス職員でもノーリスクで扱えます。
TETH

オールドレディ

O-01-04

  • 武器「孤独」: White / 拳銃。TETH拳銃三銃士の1つ。
TETH

今日は恥ずかしがり屋

O-01-12

  • 武器「今日の表情」: White / 拳銃。TETH拳銃三銃士の1つ。これら拳銃武器は攻撃の出が早く、正義ステータスが低い序盤の職員でも安定して扱いやすいのが特徴です。
